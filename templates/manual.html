AI Weather Predictor

This AI Model uses linear regression to predict the temperature based on the humidity and windspeed. You can train the model with your own data by uploading your dataset through a csv file.The model trains upon the existing dataset and the data you supplied together. The csv file should have the following column names

  • wind_speed
  • humidity
  • temperature

It is built using flask web framework and python.
The dependencies are as follows:
Flask, Pandas, Scikit-Learn.
